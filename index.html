NodeHub Help

Welcome to NodeHub!

NodeHub is designed to help you organize and reference your Blender node knowledge in a structured, visual way. Here's how to get the most out of it:

Adding Notes

Click the "Add New Note" button in the sidebar to create a new entry. Each note can include:

  • A descriptive title
  • Detailed content about what you learned
  • Category selection (Shader, Texture, etc.)
  • Tags for quick filtering
  • Optional node setup visualization text

Organizing Notes

Use the sidebar categories to filter your notes by type:

  • All Notes: View everything in one place
  • Shader Nodes: For material and shading nodes
  • Texture Nodes: Texture generation and manipulation
  • Compositing: Post-processing nodes
  • Geometry Nodes: Procedural geometry setups
  • Workflows: Useful node-based workflows
  • Cheat Sheets: Quick reference guides

Searching and Filtering

Use the search box to find specific notes by content or title. Click the "Filter" button in the toolbar to:

  • Sort notes by date or title
  • Filter by specific tags (Tips, Tutorials, Warnings, etc.)

Exporting

Click the "Export" button to save your notes collection for backup or sharing. The data will be exported as JSON which you can re-import later.

Tips for Effective Use

  • Use consistent tags to make searching easier
  • Include node setup descriptions in the visualization field
  • Date your notes to track your learning progress
  • Take advantage of the different categories to keep similar topics together
